# Utilify 🚀

Utilify is a premium, offline-first collection of **63+ browser-based utilities** designed for creators, developers, and students. Built with a focus on privacy, speed, and responsiveness, all computations are performed locally inside the user's browser memory—meaning no files or inputs are ever uploaded to a server.

🔗 **Live Platform URL:** [https://utilify.tools](https://utilify.tools)  
🏷️ **Creator Badge:** by Suyash Singh  
🛠️ **Designed & Engineered by:** Rushikesh Wagh  

---

## 🌟 Key Features

* **63+ Active Offline Utilities:** Comprehensive suite covering PDF tools, image converters, developer configs, text formatting, and financial calculators.
* **100% Client-Side Privacy:** Zero server roundtrips. Cryptography, image cropping, and PDF manipulation occur entirely client-side using WebAssembly and local JS.
* **Core Web Vitals & Speed:** Lightning-fast load times using lightweight styles, modular JS loading, and static HTML pre-rendering.
* **1000% SEO-Friendly:** Dynamic meta tags, JSON-LD Schema markup, a complete XML sitemap, and crawlable internal hash links for maximum discoverability on Google.
* **Fluid Responsive Design:** A custom glassmorphism navigation header, collapsible active sidebar directory, and responsive split columns optimized for mobile, tablet, and desktop screens.

## 🎛️ Tool Directory

### 🗂️ PDF & File Utilities
* **PDF Merger & Splitter:** Combine files or extract specific page ranges client-side.
* **PDF Page Rotator:** Rotate pages individually or in bulk.
* **Image to PDF & PDF to Image:** Convert formats and extract pages as high-resolution PNGs/JPEGs.
* **PDF Compressor:** Optimize stream layouts locally to reduce file size.
* **Protect & Unlock PDF:** Secure documents with password encryption (powered by `@pdfsmaller/pdf-encrypt-lite`) or remove locks.
* **Digital Signature Workstation:** Sign documents visually with drag-and-drop scaling, and export signed results or raw signatures.

### ⚙️ DevOps & Configuration
* **Dockerfile & Compose Generators:** Create production-ready Docker containers and stacks.
* **Nginx Configuration Builder:** General reverse-proxy and security configs.
* **AWS IAM Policy Validator:** Verify permissions schemas instantly.
* **Cron Expression Editor:** Decode, inspect, and generate cron expressions.
* **YAML ↔ JSON Converter:** Seamless schema conversions.
* **Text Diff Checker:** Side-by-side code diffing and comparisons.

### ✍️ Text & Content
* **Social Text Formatter:** Convert text to bold, italics, cursive, or gothic Unicode fonts for LinkedIn, X (Twitter), and Instagram.
* **Rich Text Editor:** WYSIWYG text styling with inline colors for Word and Google Docs.
* **Word Counter & Analyzer:** Detailed character tracking and Flesch readability indexing.
* **Smart Case Converter:** Transition strings between UPPERCASE, lowercase, camelCase, snake_case, and slug formats.

### 🎨 Design & Styling
* **CSS Box Shadow & Gradient Generators:** Visual CSS design editors with instant copy-paste code tokens.
* **Color Format Converter:** Seamless conversions between HEX, RGB, and HSL spaces.
* **Custom QR Generator:** Encode URLs and text into custom downloadable QR blocks.

### 🧮 Calculators & Math
* **Scientific Calculator:** Evaluate math, trigonometry, and logarithms offline.
* **CGPA/GPA Tracker:** Academic GPA tracking by courses and semesters.
* **EMI & Loan Repayments:** Monthly interest calculations and amortization tables.
* **GST Tax Calculator:** Calculate additions and removals of GST tax values.
* **Age & Date Difference:** Count precise birthdays and calendar day intervals.

## 🚀 Quickstart & Setup

### Prerequisites
* **Node.js** (v18.0.0 or higher)
* **npm** (v9.0.0 or higher)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Suyashrsingh/utilify.git
   cd utilify
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Launch local development server:
   ```bash
   npm run dev
   ```

4. Build production bundle:
   ```bash
   npm run build
   ```

## 🔒 Security & Privacy

Utilify is a privacy-first web application. Because it runs completely inside sandbox client-side scripts:
1. No document data, text inputs, or passwords are sent over the network.
2. The entire platform can be loaded and run fully offline (disconnected from the internet).
3. The site does not force registrations, cookies tracking, or account creation.
